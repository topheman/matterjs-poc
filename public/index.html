MatterJS-poc

The goal of this project is to make a poc of a game where you would have:

  • An editor mode to position bricks and balls
  • A runtime mode to see them fall

The physics part is based matter-js, a 2D JavaScript physics engine.

The source code is written in TypeScript and transpiled with snowpack, a new lightweight bundler for development (this poc was the occasion of testing it).

Use the following keys:

  • E: switch to editor mode
    • B: add a block on mouse position
    • C: add a circle on mouse position
    • shift+move a piece: follow magnetic grid
  • R: switch to runtime mode
  • H: show/hide Help

You can zoom in/out with the mouse wheel.

You can move the viewport.

Use the save/reload buttons to keep your work between reloading.
